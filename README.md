プログラムファイルは、以下のとおり、3種類あります。

**Complete_RAG_JTwp.pynb**  
- プロジェクトの概要説明から、プログラム処理の流れ、コードの詳細説明まで含めた、プレゼント等の説明用プログラム。
- 状況等に応じた複数の選択肢も含め、関連するすべてのコードが入った完全版。
- **注意！**：RAG構築プロセスのいくつかのポイントにおいては、複数の選択肢のなかから、その都度の目的や必要に応じた適切な手法を選択する必要がある。

**Total_RAG_JTwp.pynb**
- RAG構築のひととおりの流れを、データ収集からワークフローの構築まで通して実行可能なプログラム。
- 詳細な説明などは省き、シンプルな記述ながら、RAG構築を一気通貫で実行できる全体版。
- **注意！**：コンテンツのテキストファイル作成に3時間以上、ベクトルデータベース構築に20～45分ほどかかるため、全体で4時間ほどの時間を要する。

**Execute_RAG_JTwp.pynb**
- 最短･最小の手間･時間で手早く検索可能な状態を準備できる実行用プログラム。  
- 検索のデモや実際の検索を行うための実践版。
- **注意！**：コンテンツのテキストファイル作成やベクトルデータベース構築などのプロセスは省き、検索可能な状態を実現するための必要最小限のコードのみとなっている。
